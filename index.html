BLUG-Monitor by codetastic

This is a project run by the Blackpool Linux User Group to write a modular Server Monitor for Linux.

This project is a serious effort to create a full featured server monitoring solution, while at the same time giving an opportunity for the Blackpool LUG members to learn a new language or brush up on what they already know. Anyone wanting to contribute to the project is welcome to create a Fork and issue a Pull Request. Please use the bug tracker to identify bugs that need to be worked on, or if you have ideas for new features please submit them. Bug fixes will be looked at first before new features and there may be some delay after submitting your push request before it is responded to.
